# Iced Chocolate Matcha Latte - Scrollytelling Experience

An award-winning, immersive scrollytelling website built with Next.js 14, TypeScript, Tailwind CSS, and Framer Motion.

## Prerequisites

- Node.js 18+ 
- npm or yarn

## Installation

```bash
npm install
```

## Required Assets

### Static Cup Image
Place your high-resolution cup image at:
`/public/images/matcha-cup-static.png`

### Sequence Frames (192 frames)
Place rendered explosion sequence frames at:
`/public/images/sequence/ezgif-frame-001.jpg`
`/public/images/sequence/ezgif-frame-002.jpg`
...
`/public/images/sequence/ezgif-frame-192.jpg`

## Development
```bash
npm run dev
```
Open http://localhost:3000

## Production Build
```bash
npm run build
```
Static files will be generated in `/out` directory.

## Deployment
Deploy the `/out` folder to any static hosting:
- Vercel
- Netlify
- AWS S3 + CloudFront
- GitHub Pages

## Tech Stack
- Framework: Next.js 14 (App Router)
- Language: TypeScript
- Styling: Tailwind CSS
- Animation: Framer Motion
- Fonts: Playfair Display, Inter, Montserrat

## Features
- 192-frame canvas sequence animation
- Scroll-driven storytelling
- Interactive traveling cup animation
- Responsive design
- Premium UI/UX
- SEO optimized
- Static export ready
